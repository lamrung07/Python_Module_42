# 🐍 42 School — Python Module

> A 11-module journey through Python fundamentals, built exercise by exercise at [École 42](https://42.fr).

---

## 📚 Overview

This repository contains my solutions to the Python modules from 42's curriculum. Each module focuses on a specific layer of Python knowledge, progressing from the very basics up to object-oriented design and file handling.
---

## 📁 Repository Structure

```
.

│   ├── ex01/
│   └── ...
├── module01/

...

    └── ...
```

Each `exNN/` folder contains the source file(s) for the corresponding exercise, matching the exact file names required by the 42 norm.

---


## 🚀 Running the Exercises

Each exercise is a standalone Python script. Run any of them with:

```bash
python3 exNN/filename.py
```

No external dependencies are required — all modules use the Python standard library only.

---

## 📝 42 Norm

All code follows the [42 Norm](https://github.com/42School/norminette) coding standard:

- Python files end with a newline
- No forbidden functions used
- File and function names match the subject exactly
- Each exercise is self-contained in its own folder

These exercises were completed as part of the **Python Piscine** at [École 42](https://42.fr) — a peer-to-peer, project-based programming school with no teachers and no fixed curriculum. Every concept here was learned through doing, failing, reading, and asking peers.
